AI Powered Disease Prediction System

This intelligent medical assistant predicts possible diseases based on user symptoms using Machine Learning. It provides precautions, medication, diet plans and specialist recommendations instantly to help users take early healthcare decisions.

🚀 Project Features

🤖 AI Disease Prediction

Predicts top possible diseases using trained machine learning model based on symptoms entered by the user.

📊 Probability Analysis

Shows confidence level and probability chart for top diseases for better understanding and accuracy.

💊 Medical Recommendations

Provides medication suggestions, diet plans, workout tips and precautions for faster recovery.

👨‍⚕️ Specialist Suggestion

Recommends the right doctor or specialist based on predicted disease.

📄 Smart Medical Report

Generates a professional AI medical report with symptoms, prediction and recommendations.

⚡ Fast & User Friendly

Simple interface where users can type symptoms and get instant results within seconds.
